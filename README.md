# Grasscutter 1.0

Система управления зелёным хозяйством с интерактивной картой.

## Возможности

- 🗺️ Интерактивная карта на базе Яндекс.Карт с переключением режимов (Схема/Спутник/Гибрид)
- 🎨 Рисование полигонов на карте с сохранением в базу данных
- 📊 Система слоёв для группировки объектов
- 💾 Хранение данных в Supabase
- 📱 Адаптивный дизайн в стиле Material 3

## Технологии

- **Frontend**: Next.js 15, React 18
- **Карты**: Leaflet + Яндекс.Карты
- **База данных**: Supabase (PostgreSQL)
- **Анимации**: Lottie React
- **Стили**: CSS-in-JS
